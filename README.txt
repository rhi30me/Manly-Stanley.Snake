# Manly Stanley
A rainbow snake game built with Python and Pygame. Eat the flashing stars, grow longer, and don't die!!!!! ! N0 M0RE W@R | END US AGGRESSION

## Features
Rainbow-colored snake that grows as you eat, star-shaped food in randomized colors, generated randomly throughout the grid you play on. Animated snake eyes that follow your direction, speed increases every 5 points, High score tracking (for the most part )

## Requirements
You will need Python 3.8 or higher and the Pygame library installed on your machine.


## Installation
Start by cloning the repository to your local machine.

,,
git clone https://github.com/YOUR_USERNAME/manly-stanley-snake.git
cd manly-stanley-snake
''

Next, install Pygame using pip.

,,
pip install pygame
''

If you are on macOS and using Homebrew Python, use this command instead.

,,
/opt/homebrew/bin/python3 -m pip install pygame --break-system-packages
''

## Running the Game
Once Pygame is installed, you can launch the game with the following command.

,,
python3 snake_game.py
''
If you are on macOS with Homebrew Python, run it like this instead.

,,
/opt/homebrew/bin/python3 snake_game.py
''

## How to Play
Use the arrow keys or W A S D to move Manly Stanley around the screen. Press Space to start or restart the game at any time, and Q is quit. Eat the flashing stars to grow longer and earn points. Avoid running into the walls or your own tail or it's game over. The snake gets faster every 5 points.
