# Manly-Stanley.Snake
Manly Stanley is a colorful rainbow snake game built with Python and Pygame. Eat the flashing stars, grow longer, and have fun :) You will need Python 3.8 or higher and the Pygame library installed on your machine. Once Pygame is installed, you can launch the game with the following command. python3 snake_game.py.
